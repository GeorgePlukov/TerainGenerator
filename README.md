Random Terrain Generation 
======
This projects purpose was to create a square grid based randomly generated map. 

Current block types:
   
   - Grass
   - Sand
   - Water

Current Terrain options:

   - Lakes

### Change Log
#####Version 1.0
   - Added to github
   - Functioning lakes
   - Terrain being generated
   - Variable map sizes
   - No more 'square map syndrome'
